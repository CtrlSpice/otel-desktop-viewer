Release Checklist:

1. Create new release with GoReleaser
   - Use GoReleaser's --snapshot flag to test the release process
   - Verify all builds complete successfully
   - Check that artifacts are generated correctly

2. Update Homebrew formula
   - Let GoReleaser automatically update the formula with correct SHA256 checksums
   - Verify the formula changes in the homebrew-otel-desktop-viewer repository

3. Test the formula locally
   - Install from the local formula to verify it works
   - Test on both Intel and Apple Silicon Macs if possible

4. Push the updated formula
   - Commit and push the updated formula to the homebrew-otel-desktop-viewer repository
   x Verify the formula is accessible via brew install

Note: Token permissions verified - current GITHUB_TOKEN has sufficient permissions for all operations.
